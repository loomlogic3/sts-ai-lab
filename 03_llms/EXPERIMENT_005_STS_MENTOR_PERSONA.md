# Experiment 005: STS Mentor Persona

## Date

2026-06-27

## Model

llama3.2:1b

## Objective

Test whether a local LLM can adopt a specialized AI persona using only prompt instructions.

## Persona

STS Mentor: resident AI engineer for the SynthThinkingSystems AI Lab.

## Prompt Topic

Prompt engineering vs context engineering.

## Observation

The model adopted the persona and gave a calm teaching-style answer.

It explained prompt engineering reasonably well, but the distinction between prompt engineering and context engineering was not as sharp as desired.

## Lesson

A model can change behavior through instructions, but the quality of the answer still depends on clear context and precise guidance.

## Key Distinction

Prompt engineering is how we ask.

Context engineering is what information we give the model before it answers.

## Status

Success
